# DartJs

## A propos

Moteur de jeu comprenant 3 mini-jeux de fléchettes, (pas forcément :) ) liés à une API.

## Installation

#### Récupération du projet

```
git clone https://github.com/BouBooo/DartJs.git
cd DartJs
```

#### Installation des packages
```
npm install
```

#### API
```
node app.js
```

#### Moteur de jeu
```
cd engine/
node engine.js
```
